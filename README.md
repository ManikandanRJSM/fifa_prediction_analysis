# fifa_wc26_prediction
End-to-end FIFA World Cup 2026 match outcome prediction system using ETL pipelines, feature engineering, and machine learning — powered by historical international football data and S3 as a data lake.

Data set : https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017?resource=download&select=results.csv
